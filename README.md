# GoodGameGuides

#### GoodGameGuides is a GoodReads clone built by Alex Smaldone, Laura Maser, and Whitney Minson. This was our first full stack project at App Academy, and one of our only projects built using vanilla Javascript. 
#### Check out our clone at the live site! [GoodGameGuides](https://good-game-guides.herokuapp.com/)

## Technologies used:

Javascript | Node.js | Express | Sequelize | CSS | Pug | Heroku
